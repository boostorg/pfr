Function template flat_read

boost::pfr::flat_read

Synopsis

// In header: <boost/pfr/flat/io.hpp>


template<typename Char, typename Traits, typename T> 
  void flat_read(std::basic_istream< Char, Traits > & in, T & value);

Description

Reads flattened POD value from stream in

Note: If reflecting structures with enums may reinterpret_cast enum to a reference to underlying type.

Example: 

struct my_struct { int i, short s; };
my_struct s;
std::stringstream ss;
ss << "{ 12, 13 }";
ss >> s;
assert(s.i == 12);
assert(s.i == 13);
