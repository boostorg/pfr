Function get

boost::pfr::get — Returns reference or const reference to a field with index I in aggregate T.

Synopsis

// In header: <boost/pfr/precise/core.hpp>


template<std::size_t I, typename T> 
  decltype(auto) constexpr get(const T & val);
template<std::size_t I, typename T> decltype(auto) constexpr get(T & val);

Description

Requires: C++17 or C++14 flat POD or C++14 with not disabled Loophole.

Example: 

struct my_struct { int i, short s; };
my_struct s {10, 11};
assert(boost::pfr::get<0>(s) == 10);
boost::pfr::get<1>(s) = 0;
