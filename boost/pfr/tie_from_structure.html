Function template tie_from_structure

boost::pfr::tie_from_structure — Create a tuple of lvalue references capable of de-structuring assignment from fields of an aggregate T.

Synopsis

// In header: <boost/pfr/precise/core.hpp>


template<typename... Elements> 
  unspecified tie_from_structure(Elements &... args);

Description

Example: 

auto f() {
  struct { struct { int x, y } p; short s; } res { { 4, 5 }, 6 };
  return res;
}
auto [p, s] = f();
tie_from_structure(p, s) = f();
