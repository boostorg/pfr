Global tuple_size_v

boost::pfr::tuple_size_v — tuple_size_v is a template variable that contains fields count in a T and works for any T that supports aggregate initialization even if T is not POD. Flattens only multidimensional arrays.

Synopsis

// In header: <boost/pfr/precise/tuple_size.hpp>

constexpr std::size_t tuple_size_v;

Description

Requires: C++14.

Example: 

std::array<int, boost::pfr::tuple_size_v<my_structure> > a;
