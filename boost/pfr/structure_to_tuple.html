Function template structure_to_tuple

boost::pfr::structure_to_tuple — Creates an std::tuple from an aggregate T.

Synopsis

// In header: <boost/pfr/precise/core.hpp>


template<typename T> constexpr auto structure_to_tuple(const T & val);

Description

Requires: C++17 or C++14 flat POD or C++14 with not disabled Loophole.

Example: 

struct my_struct { int i, short s; };
my_struct s {10, 11};
std::tuple<int, short> t = make_tuple(s);
assert(get<0>(t) == 10);
