Function template write

boost::pfr::write — Writes aggregate value to out

Synopsis

// In header: <boost/pfr/precise/io.hpp>


template<typename Char, typename Traits, typename T> 
  void write(std::basic_ostream< Char, Traits > & out, const T & value);

Description

Requires: C++17 or C++14 constexpr aggregate intializable type.

Misc: Reinterpret casts unrelated types in C++14 only. May have overhead in C++14 only on non-optimizing compilers.

Example: 

struct my_struct { int i, short s; };
my_struct s{12, 13};
write(std::cout, s); // outputs '{12, 13}'
